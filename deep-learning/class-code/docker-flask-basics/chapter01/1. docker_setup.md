# docker 개발 환경 구축 명령어

### 1. GPU 가속 딥러닝 환경 구축
```
docker run -it -d --gpus all -e DISPLAY=host.docker.internal:0.0 --name tensorflow -v /tmp/.X11-unix:/tmp/.X11-unix -v d:\DeepLearning_Class:/workspace tensorflow/tensorflow:2.16.2-gpu /bin/bash
```

- `-it` : 대화형 터미널 사용
- `-d` : 백그라운드에서 실행
- `--gpus all` : 호스트의 모든 GPU 자원 할당
- `-e DISPLAY` : 호스트의 GUI 출력 장치와 연결
- `-v` : 호스트 폴더 (d:\DeepLearning_Class)를 컨테이너 내부(/workspace)와 공유

### 2. 컨테이너 내부용 패키지 관리 및 GUI 도구 설치

#### - 시스템 패키지 목록 갱신
```
apt update
```

#### - Thinter 설치
```
apt install -y python3-tk
```

#### -  디지털 시계 도구 dclock 설치
```
apt install -y dclock
```

#### -  Git 설치 (추가)
```
apt install -y git
```
