# Flask용 Container 생성

### 1. 컨테이너 생성 및 실행
```
docker run -d -it -p 5000:5000 --name flaskserver -v d:\DeepLearning_Class\chapter02\flaskserver:/workspace python:3.12-slim
```
- `-d` : 컨터이너를 백그라운드에서 실행하는 옵션
- `-it` : 컨테이너 내부 터미널과 상호작용 할 수 있도록 설정
- `-p 5000:5000` : 호스트 윈도우의 5000번 포트와 컨테이너의 5000번 포트를 연결 (포트 포워딩)
- `--name` : 생성할 컨테이너 이름을 flaskserver 로 지정
- `-v d:\DeepLearning_Class\chapter02\flaskserver:/workspace` : 볼륨 마운트 설정으로 내 컴퓨터의 특정 폴더를 컨테이너 내부의 /workspace 폴더와 실시간 공유
- `python:3.12-slim`: 가벼운 버전의 파이썬 3.12 이미지를 기반으로 컨테이너 생성 

### 2. 컨테이너 내부 접속
```
docker exec -it flaskserver /bin/bash
```
- 실행 중인 `flaskserver` 컨테이너의 터미널에 접속하여 명령어를 입력할 수 있는 상태로 연결

### 3. 패키지 설치
1. 컨테이너 내부의 파이썬 환경에 Flask 프레임워크 설치
```
pip install flask
```

2. 버전 확인
```
flask --version
```
