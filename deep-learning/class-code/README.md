# Deep Learning Class Code

딥러닝/생성형 AI 수업 시간에 사용한 Docker, Flask, Local LLM, LangChain,
ComfyUI, 시계열 딥러닝 코드를 모은 폴더입니다.

## Layout

- `docker-flask-basics/`: Docker, WSL, Flask 기본 수업 자료
- `flask-api-basic/`: Python API 서버 진입 예제
- `flask-web/`: Flask 웹 예제
- `local-llm/`: Ollama와 Local LLM API 예제
- `langchain-basic/`: LangChain 기본 구조와 chain 예제
- `langchain-memory-basic/`: LangChain memory 예제
- `comfyui-basic/`: ComfyUI API 호출 예제
- `time-series/`: 시계열 딥러닝 노트북과 windowing 예제

과제와 텀프로젝트 앱은 루트의 `lab/deep-learning/` 아래에 따로 정리했습니다.
