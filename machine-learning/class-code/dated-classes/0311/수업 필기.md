## 2. 확률

모든 경우의 수와 각 사건의 발생 확률 등을 말함

- ### 사건
**단순 사건이라고 주사위를 볼 때**
1. `홀수`의 확률은 1 , 3, 5 로 1/2 로 `50%` 의 확률을 가진다고 할 수 있다. 
2. `짝수`의 확률 또한 2, 4, 6 으로 1/2로 `50%` 의 확률을 가진다고 할 수 있다.
3. `짝수` 가 나오지 않을 확률이나 `홀수`가 나오지 않을 확률 또한 1/2 로 `50%` 의 확률을 가진다고 할 수 있다.

**복합 사건이라고 주사위를 볼 때**
1. 홀수이거나 3보다 큰 두 조건을 만족하는 복합사건의 확률의 경우, 3보다 큰 조건은 4,5,6 3지인데 홀수이여야하므로 `5` 한가지만 가능하다. 따라서 확률은 1/6이다.

**독립 사건 이라고 주사위를 볼 때**
1. 주사위를 두개를 굴리는 경우, 두 주사위는 다른 주사위 값들이 가진 확률을 변하게 하지 않으므로 두 사건은 독립이 된다.
2. 두 주사위의 합이 2인 확률은?
3. 두 주사위의 합이 3인 확률은?

- ### 조건부 확률
1. U1 항아리에 빨간 공 3개와 파란 공 1개
2. U1 항아리에 빨간 공 2개와 파란 공 2개
3. 동전을 던져서 앞면이면 U1 항아리에서 공 하나를, 뒷면이면 U2 항아리에서 공 하나를 꺼낼때 U1에서 빨간 공을 꺼낼 확률
